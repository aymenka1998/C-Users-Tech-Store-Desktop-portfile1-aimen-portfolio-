# 🚀 Aimen Portfolio | Full-Stack Developer

أهلاً بك في مستودع مشروع معرض أعمالي الشخصي. تم بناء هذا الموقع باستخدام أحدث تقنيات الويب الحديثة لضمان الأداء العالي والتجربة الممتازة للمستخدم.

## 🛠 التكنولوجيات المستخدمة (Tech Stack)

* **Frontend:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Framer Motion](https://www.framer.com/motion/) للحركات.
* **Backend:** [Strapi CMS](https://strapi.io/) (Headless CMS)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Deployment:** Vercel (Frontend) & Railway/Render (Backend)

## ✨ المميزات الرئيسية

* **Vibe Coding Optimized:** تم تطوير الكود باستخدام أدوات الذكاء الاصطناعي الحديثة (Cursor, v0) لضمان نظافة المعمارية وسرعة التنفيذ.
* **Dynamic Projects:** يتم جلب جميع المشاريع وتفاصيلها ديناميكياً من خلال Strapi API.
* **Contact System:** نظام مراسلة متكامل يربط بين Next.js و Strapi.
* **Responsive Design:** متوافق تماماً مع جميع الشاشات (Mobile, Tablet, Desktop).
